## Laravel ORM Manager

Laravel ORM Manager is an attempt to create a re-usable admin panel for Laravel, in the same vein as Django's default 
admin console.

#### Dependencies:
- Laravel
- Vue for the frontend - can be optional in the future
- DBAL for model metadata retrieval
